# Amanda — Landing (Bootstrap)

Landing page de apresentação com Bootstrap, foco em responsividade, UX e um formulário “demo” (salva mensagens no LocalStorage + inbox em modal).

## Demo (Vercel)

- https://amanda-landing-apresentacao.vercel.app/

## Rodar localmente

### Opção 1) Abrir direto

1. Baixe/clonar o projeto
2. Abra o arquivo `index.html` no navegador

> Observação: algumas funções (ex: copiar para área de transferência) podem funcionar melhor em `localhost` do que em `file://`.

### Opção 2) Servidor local

**Com Node (npx):**

**\*Ou
VS Code (Live Server / Go Live)**

```bash
npx serve
```
